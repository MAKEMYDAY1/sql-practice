# Визуализация метрик пользователей

## Описание дашборда

### График 1: Ежедневные новые пользователи и курьеры

- **Тип**: Линейный график
- **Метрики**:
  - new_users (синяя линия)
  - new_couriers (красная линия)
- **Назначение**: Отслеживание динамики привлечения новых пользователей

### График 2: Накопительный рост пользователей и курьеров

- **Тип**: Линейный график
- **Метрики**:
  - total_users (синяя линия)
  - total_couriers (красная линия)
- **Назначение**: Анализ общего роста пользовательской базы
